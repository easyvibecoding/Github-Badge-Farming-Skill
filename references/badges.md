# GitHub Badges 詳細規則 & 等級（2026）

## 徽章等級系統

部分徽章有分 Bronze / Silver / Gold / Platinum 四個等級。

---

## ⚡ Quickdraw

- **條件**：Issue 或 PR 在開啟後 5 分鐘內關閉
- **等級**：無分級（一次即可）
- **注意**：可以關閉自己開的 Issue

---

## 😅 YOLO

- **條件**：合併一個完全沒有 Code Review 的 PR
- **等級**：無分級（一次即可）
- **注意**：在自己的 Repo 有管理員權限時，直接 merge 即可觸發

---

## 🦈 Pull Shark

- **條件**：合併 PR（不限在哪個 Repo）
- **等級**：

| 等級 | 需要合併 PR 數 |
|---|---|
| Bronze | 2 |
| Silver | 16 |
| Gold | 128 |
| Platinum | 1024 |

- **注意**：PR 必須是你自己開的，且被 merge（不限誰 merge）

---

## 🤝 Pair Extraordinaire

- **條件**：PR 的 commit 中有有效的 `Co-authored-by`，且該 PR 被 merge
- **等級**：

| 等級 | 需要次數 |
|---|---|
| Bronze | 1 |
| Silver | 10 |
| Gold | 24 |
| Platinum | 48 |

- **重要**：`Co-authored-by: Name <email>` 中的 email 必須是真實 GitHub 帳號的 commit email
- 可在 GitHub → Settings → Emails 查詢自己的 commit email

---

## 💖 Public Sponsor

- **條件**：透過 GitHub Sponsors 公開贊助任何開發者/組織
- **等級**：無分級（一次即可）
- **最低花費**：找支援 Custom Amount 的帳號，$1 即可

---

## 🧠 Galaxy Brain

- **條件**：在 Discussions 中，你的回答被問題發起者標記為 Accepted Answer
- **等級**：

| 等級 | 需要被採納次數 |
|---|---|
| Bronze | 2 |
| Silver | 8 |
| Gold | 16 |
| Platinum | 32 |

- **2024 規則更新**：
  - ❌ 自己採納自己的回答不算
  - ✅ 必須由他人（提問者）採納
  - ❌ 不再支援 GitHub Community Discussions（官方論壇）
  - ✅ 仍支援其他開啟 Discussions 功能的 Repository

---

## ⭐ Starstruck

- **條件**：你建立的 Repository 達到指定 Star 數
- **等級**：

| 等級 | 需要 Star 數 |
|---|---|
| Bronze | 16 |
| Silver | 128 |
| Gold | 512 |
| Platinum | 4096 |

- **注意**：互刷 Star 違反 GitHub ToS

---

## ❄️ Arctic Code Vault Contributor（已無法取得）

- 曾為 2020 GitHub Archive Program 的 Repo 貢獻程式碼
- 現在無法再取得

---

## 🚁 Mars 2020 Contributor（已無法取得）

- 曾為 Mars 2020 火星直升機任務的 Repo 貢獻程式碼
- 現在無法再取得

---

## 膚色客製化

部分徽章（Quickdraw、Pair Extraordinaire 等）的 emoji 圖示會根據你在 GitHub 設定的 Emoji Skin Tone 偏好變化。
設定路徑：GitHub → Settings → Appearance → Emoji skin tone preference
